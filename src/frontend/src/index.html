Mass-Subcatchment calculation
Several catchment areas can be calculated in one go. To do this, a point shapefile (LV95) must be created with the points for which the catchment areas are to be calculated. Pack all the files belonging to the shapefile together as a ZIP file and upload them here. All catchment areas are calculated and summarised in one shapefile. After successful calculation, the file can be downloaded as a ZIP.
